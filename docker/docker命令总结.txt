docker pull <image-name>   下载镜像
docker push                上传镜像
docker images              显示镜像列表
docker commit              将容器保存为镜像
docker build <docker-file> 构建镜像
docker history             显示镜像构建历史
docker tag                 重命名镜像
docker rmi                 删除镜像
docker search              搜索镜像

docker run                 启动容器，如果本地没有先下载镜像




docker ps 或者 docker container ls 显示容器正在运行
